# Steelhead data

## Overview

This repository contains raw data, code, and output datasets used to inform population indicators in the [Pacific Salmon Explorer](https://salmonexplorer.ca/) for steelhead Conservation Units in coastal and southern British Columbia, including the Columbia region. 

Note that data for the Northern Transboundary region are compiled in [transboundary-data](https://github.com/salmonwatersheds/transboundary-data).

## Files

* `data` contains raw data files. We attempt to save files 'as provided'. Details of data sources are in the `data/README.md`
* `code` contains R scripts to load raw data files and massage them into a format that can be ingested into the database behind the Salmon Explorer.
* `output` contains the most recent version of datasets in the Salmon Explorer. Past versions are archived in PSF's internal files and can also be viewed by looking at past versions of this repo.
